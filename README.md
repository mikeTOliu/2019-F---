# Railgun
***2019全国电赛作品***  

小组成员 

[mikeTOliu](https://github.com/mikeTOliu) 

[clyx-mm](https://github.com/clyz-mm) 

[wxp12](https://github.com/wxp12)


- main.py  
openmv程序，可直接上传到openmv闪存中。  
- test_led.ino、TM1638.ino  
arduino程序，可直接烧录。

- 功能  
可通过ledkey输入距离和角度，一键启动后开炮。  
可通过按键一键启动后自动追踪目标并开炮。  
可摇摆开炮。
